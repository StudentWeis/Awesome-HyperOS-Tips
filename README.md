# Awesome-HyperOS-Tips



HyperOS 使用小技巧。

## 锁屏

| 类型 | 方法                                         | 备注                       |
| ---- | -------------------------------------------- | -------------------------- |
| 操作 | 锁屏左滑可以进去控制中心打开闪光灯。         | 临时想用闪光灯不需要解锁。 |
| 设置 | 可以设置锁屏时可下拉操作，可以加上控制中心。 |                            |

| 类型 | 方法                                                            | 备注               |
